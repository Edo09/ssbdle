# Super Smash Bros. Ultimate: Website Design Guide

To capture the electric, high-energy atmosphere of *Super Smash Bros. Ultimate*, a **dark-mode aesthetic** is highly recommended. The official artwork is packed with intense, saturated character colors; using a deep, muted background prevents the website from looking cluttered and allows the vibrant elements to truly pop.

Here is a curated professional color palette extracted directly from the artwork's core elements and the official website's branding.

## The Smash Ultimate Palette

| Color Role | Hex Code | Visual Origin | Best Used For |
| :--- | :--- | :--- | :--- |
| **Primary Background** | `#0D1117` | The deep, dark space in the outer corners | Main page backgrounds, sections, and footers. |
| **Card / Surface Background** | `#161B22` | The mid-tone shadows behind the characters | Container boxes, character cards, and navigation bars. |
| **Smash Crimson (Accent 1)**| `#DE1A22` | Mario's iconic red & classic Nintendo UI | High-priority buttons, hover states, and active tabs. |
| **Chrono Blue (Accent 2)** | `#009BFF` | Link’s Champion Tunic and the energy beams | Secondary buttons, links, categories, and borders. |
| **Energy Cyan (Highlight)** | `#8CEFFF` | The blinding central cross-star blast | Text highlights, glow effects, or small badge elements. |
| **Text Light** | `#F0F6FC` | The brilliant light crests | Primary headings and readable body text. |

## UI/UX Implementation Strategy

* **The Foundation (Dark Canvas):** Set your body background to `#0D1117`. Gaming websites thrive on dark themes because they mimic a cinematic or console-like experience. Use `#161B22` for UI cards to give the layout depth.
* **The Clash of Elements (Red vs. Blue):** Use **Smash Crimson** and **Chrono Blue** to create a natural competitive contrast. For example, if you feature character matchups, use Red for "Player 1 / Attack" elements and Blue for "Player 2 / Defense" elements.
* **Typography Contrast:** For the text, avoid pure `#FFFFFF` for long paragraphs to prevent eye strain against the dark background. Instead, use the softer `#F0F6FC` for body text, and save pure white paired with a subtle **Energy Cyan** text-shadow for your massive headings.
* **Vibrant Accents:** Keep the bright neon colors (like the Cyan or pure Red) reserved strictly for interactive components (like a button hover effect). If too many bright colors are used across the background, the site will quickly feel overwhelming.
